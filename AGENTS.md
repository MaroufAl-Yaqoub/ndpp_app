\# NDPP Flutter App - Codex Instructions



\## Project Overview

This is a Flutter mobile application.



\## Setup

Run:

flutter pub get



\## Run App

flutter run



\## Build

flutter build apk



\## Project Structure

\- lib/ → main app code

\- models/ → data models

\- screens/ → UI screens

\- services/ → API services

\- theme/ → app styling



\## Rules

\- Keep code clean and readable

\- Use proper naming conventions

\- Do not break existing functionality

\- Always test before finalizing



\## Notes

\- API configs are in core/config/

\- Mock data is used in some parts
